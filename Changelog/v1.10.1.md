# Drive-Future Backlink Manager - Changelog v1.10.1

**Date:** 2026-03-16

### Overview
Fixed a critical bug that caused bulk Domain Metrics uploads (Excel/CSV) to fail when processing formatted strings from typical SEO tools (e.g., Ahrefs, Moz). 

### Backend & Core Logic
- `src/lib/utils.js`:
  - Added the `parseMetric(val, isFloat)` heuristic utility to gracefully strip commas, percentage markers (`%`), and translate magnitude postfixes (`K`/`M`) into raw numeric values.
  - Correctly evaluates generic missing metric markers (`N/A`, `NULL`, `-`) directly into database `null`.
- `src/app/admin/domains-manager/actions.js`:
  - Patched `uploadDomainMetrics` and `saveDomains` Server Actions to utilize `parseMetric` prior to routing payload data points through `zod` coercion logic to prevent the pipeline from failing on `NaN` resolutions.
