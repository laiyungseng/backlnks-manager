# Changelog - v1.4.0

## [2026-03-03]

### Added
- **Schema Builder**: Brand new admin page (`/admin/schema-builder`) featuring a visual database table designer.
- **SQL Code Generator**: Real-time SQL syntax generation based on user inputs, including data types, defaults, nullable toggles, and UUID generation strings.
- **Direct Execution API**: Secure action to ping Supabase APIs directly from the browser payload using an encrypted Management API key stored in Settings.
- **Warning Guards**: The Schema Builder now intelligently blocks execution and prompts for credential inputs if API keys are missing.
