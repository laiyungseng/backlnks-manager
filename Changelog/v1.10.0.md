# Drive-Future Backlink Manager - Changelog v1.10.0

## Phase 35: Domain Metrics Bulk Upload via Excel/CSV

**Date:** 2026-03-14

### Overview
Implemented the ability for administrators to upload Excel/CSV files directly into the Domains Manager to bulk-update metrics (Domain Rating, Traffic, Domain Age, Spam Score) for existing domains, or to bulk-insert new domains.

### Frontend Changes
- **Dependency Added**: Integrated `xlsx` package to support robust parsing of spreadsheet files.
- `src/app/admin/domains-manager/page.jsx`:
  - Added a responsive file upload button.
  - Implemented `handleFileUpload` with frontend spreadsheet-to-JSON parsing algorithms.
  - Added custom heuristic column mapping to intelligently extract headers regardless of case or alias (e.g., `DR` vs `Domain Rating`).
- `src/app/vendor/[vendor_name]/[hash]/VendorForm.jsx`:
  - Extracted inline URL parsing logic into a reusable utility file.

### Backend & Core Logic
- `src/lib/utils.js`:
  - Created new shared utility library.
  - Deployed `parseDomainUrl` for centralized, strict domain footprint resolution.
- `src/app/admin/domains-manager/actions.js`:
  - Implemented the `uploadDomainMetrics` Server Action.
  - Built a bulk chunking Upsert pipeline fetching existing URL identifiers to persist relational mapping dynamically.
  - Bound incoming metrics directly against strict `zod` coercion validation schema `domainSchema`.
  - Added zero-downtime overwrite protection mapping untouched attributes effectively while automatically re-stamping `last_checked_at` with a fresh ISO timestamp.
