# Changelog (Latest Update: UX & Auth Polish)

## Added
- **Database Login Auth:** Added SQL migration `phase22_admin_users.sql` that provisions an `admin_users` table securely alongside `loginRouter` endpoints.
- **Login Tracking:** Integrated `last_login` TIMESTAMPTZ into `admin_users` to log precisely when an admin accessed the system.
- **Success Actions:** New Project dashboard now jumps to top with a browser alert upon successful kickoff creation.
- **Sign Out Actions:** Full session clearance now attached to the global sidebar `Sign Out` button.

## Fixed
- **Instant Cache Drop:** Admin Server Actions (`finalizeProject`, `deleteProject`, `toggleProjectLock`) now forcefully invoke NextJS `revalidatePath('/admin', 'layout')` causing zero-refresh UI syncs for all actions.
- **Optimistic Table Removal:** Dashboard `delete` functionality hooks into local `setProjects` for immediate UI response.
- **Bulk Save Bug:** Glide Data Grid's aggressive synchronous `onCellEdited` event vomiting was circumvented with a fully custom React `onPaste` 2D-mapping loop in `VendorForm.jsx`. `isDirty` now correctly trips to save 1000s of rows simultaneously via the 1-second auto-timer.
