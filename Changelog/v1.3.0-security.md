# Changelog - v1.3.0

## [2026-03-03]

### Added
- **Global Admin Guard (Middleware)**: New `src/middleware.js` to intercept and protect all routes under `/admin/*`. Unauthorized direct URL access now triggers an immediate redirect to the login page.
- **Server-Side Session Cookies**: Updated the login flow to set a secure `df_admin_session_active` cookie, enabling server-side authentication verification.
- **Git Privacy Policy**: Updated `.gitignore` to exclude `MEMORY.md` from version control, preserving local development context logs.

### Changed
- **Authentication Flow**: Login page now synchronizes client-side `localStorage` with server-side cookies for a holistic security model.

### Fixed
- **URL Path Exploits**: Closed security hole where privileged dashboard directories could be accessed via direct path entry in the browser address bar.
