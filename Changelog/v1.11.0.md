# Changelog v1.11.0

## [1.11.0] - 2026-03-19

### Added
- **Vendor Portal: Extra Placements**: Added "Add Extra Placement" button allowed vendors to insert new rows.
- **Vendor Portal: Editable Core Fields**: Enabled editing of `Target URL`, `Anchor Text`, and `Language` for extra rows.
- **Lucide Icons**: Added `PlusCircle` icon for the add-on feature.

### Changed
- **Grouped Language Distribution**: Modified the vendor project logic to group rows by language based on the order they were added during the kickoff project creation (no longer interleaved).

### Fixed
- **Language Preservation**: Fixed a bug in the placement processor where individual row languages were being overwritten by the project-wide default language during normalization/finalization.

### Security
- Maintained `isLocked` guard for finalized projects, preventing extra rows from being added to completed projects.
