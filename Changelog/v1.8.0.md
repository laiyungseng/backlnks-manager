# Drive-Future Backlink Manager - Changelog v1.8.0

## Phase 33: Vendor Portal UI Enhancements

**Date:** 2026-03-10

### Overview
Improved the user experience in the Vendor Portal by allowing text selection and copying within readonly grid columns.

### Frontend Changes
- `src/app/vendor/[vendor_name]/[hash]/VendorForm.jsx`: Modified the `@glideapps/glide-data-grid` configuration to set `allowOverlay: true` on readonly columns (`domain_url`, `target_url`, `anchor_text`, `language`, `published_date`). 
- Changed `target_url` kind from `Uri` to `Text` to allow text selection and copying while preventing the browser from redirecting to the link on double-click.
- Maintained `readonly: true` across all relevant columns to ensure data integrity.
