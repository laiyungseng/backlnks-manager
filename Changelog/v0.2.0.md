# Changelog v0.2.0

## Features & Structural Architecture Restructure
* **Dynamic Form Generation:** `src/app/admin/new-project/page.jsx` rewritten using React State mappings to mathematically enforce and distribute 1-to-many Target/Anchor mappings against total Subproject Quota.
* **Database Architecture Upgrade:** Implemented Phase 2 Database redesign. Dropped statically fixed columns from `projects` and pushed a 1-to-many geometry table `project_targets`.
* **Virtual Vendor Placements (Staging Data):** Supabase pipeline updated to inject unverified Vendor submissions strictly as JSONB array structures securely inside the `project_list` table row instead of directly mapping and hitting relational restrictions.
* **Country & Language Flags:** Added specific meta trackers to Kickoff logic to route correctly to isolated table formats.
* **Cryptographic Hash Logic:** Sunset the standard UUID mapping mechanism and introduced direct `SHA-256` hash generations per-project to generate impenetrable, unpredictable external Vendor Portal Links.
* **Dynamic Admin UI Mapping:** Overhauled `src/app/admin/placements/page.jsx` to render real-time aggregated Target Domain summaries (e.g. 2 / 5 Uploaded) rather than deeply granular individual URL rows, significantly improving high-level Dashboard readability.
## Vendor UI Optimizations & Global Drip Feed (Phases 5-9)
* **High-Density Spreadsheet UI:** Completely removed card-based padding from the Vendor Portal in favor of a native Excel-style locked-header table.
* **Auto-save & Timestamps:** Added a 1-second debounce auto-save trigger on the URL field that writes precise ISO `published_date` stamps to the database.
* **Smart Multi-line Paste:** Vendors can now copy a massive column of URLs from Excel/Sheets and paste them directly into a single cell, auto-flowing downwards into empty slots.
* **Analytics Overlays & Filtering:** Added a dynamic Project Progress tracker grouped automatically by Anchor Text. Added global column filtering using React-Lucide toggle components.
* **Global Drip Feed Mode:** Extended the Kickoff form with a brand new "Drip Feed" enabler. Projects can now be strictly scheduled over X Days.
* **Active Quota Tracking:** Admin Dashboard now displays live Dripfeed Statuses (URLs/day), and the Vendor Portal features a "Smart Align" amber banner enforcing the specific "Today's Quota" limit against the saved timestamps.

## Glide Data Grid Performance Revamp (Phase 15)
* **High-Performance Grid Component:** Swapped out the old HTML table constraint in `VendorForm.jsx` for `@glideapps/glide-data-grid` to elegantly handle 10k-100k rows without DOM bloat.
* **Native Spreadsheet UX:** VENDOR Portal now natively supports dragging matrices, multi-line Excel copy-pasting, and instantaneous typing overrides (via floating portal text editors).
* **Grid Bug Fixes:** Refactored SSR hydration mismatches, CSS parent flex constraints, and state mutation memoization that previously caused the text editor overlays to vanish mid-keystroke.
