# v1.5.0 — 2026-03-05

## Projects Table JSONB Reconstruction (Phase 27)

### Summary
Refactored the `projects` table from a flat-column structure to a JSONB-based schema. All project metadata now lives inside a `project_details` JSONB array column, enabling easier schema evolution and nested field access.

### SQL Migration
- **File:** `Changelog/phase27_projects_table_reconstruction.sql`
- Added top-level columns: `user`, `created_date`, `complete_date`, `project_details`
- Migrated all existing flat column data into `project_details[0]`
- Converted `languages` array to `{"languages-ratio": [{"lang-code": "...", "ratio": ...}]}`
- Dropped 23 legacy flat columns
