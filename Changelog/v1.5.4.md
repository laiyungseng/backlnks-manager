# Changelog v1.5.4

## [v1.5.4] - 2026-03-06

### Added
- **Domain Manager Auto-Timestamp**: `Last_checked_at` now automatically updates to the current time whenever a row is added or edited in the Domain Manager.
- **Smart JSONB Merging**: Server actions in Vendor and Domain managers now perform a shallow merge of new edits with existing JSONB data, preventing accidental data loss of non-edited fields (like Contact, DR, Traffic).

### Fixed
- **JSONB Truncation**: Resolved issue where JSONB columns were being overwritten with minimal templates during project finalization and manual saves.
- **Schema Cache Errors**: Removed legacy column references from the Normalization Engine that were causing insertion crashes after the migration.
- **Duplicate Domain Handling**: Enhanced `placementProcessor.js` to fetch and match existing domains by hostname before attempting insertion, preventing unique constraint violations.

### Technical Details
- Refined `placementProcessor.js` with robust error logging and deterministic domain matching.
- Updated `DomainManager.jsx` with real-time UI updates for timestamps and improved filtering.
